        ** Query Selector we are going to use are :-

        document.querySelector('h1') --> Gives the 1st h1 in the web page.
        document.querySelector('h2') --> Gives the 1st h2 in the web page.

        Just as Css in this also we can select class using "." and id using "#" inside queryselector

        document.querySelector('#title') --> Gives the result of the id having name "title".
        document.querySelector('.heading') --> Gives the result of the class having name "heading".

        We can also use more selectors just as we use in Css selectors.
        for Ex:-
        document.querySelector('input')
        document.querySelector('input[type="password"]')       To select any attribute of input we can put that in square brackets.

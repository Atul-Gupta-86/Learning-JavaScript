** Query Selector we are going to use are :-

document.querySelector('h1') --> Gives the 1st h1 in the web page.
document.querySelector('h2') --> Gives the 1st h2 in the web page.

Just as Css in this also we can select class using "." and id using "#" inside queryselector

document.querySelector('#title') --> Gives the result of the id having name "title".
document.querySelector('.heading') --> Gives the result of the class having name "heading".

For getting all element having that class or id we can do document.querySelectorAll('..heading_23') --> Gives Node list or for understanding you can 
say psudo/False/Partial array (Not an pure Array) of (here) two result as in this(our) page we give two element with class heading_23 so we get two 
to access any particular we can do document.querySelectorAll('..heading_23')[0]/[1] as per our requirement.

We can also use more selectors just as we use in Css selectors.
for Ex:-
document.querySelector('input')     to select input element
document.querySelector('input[type="password"]')       To select any attribute of input we can put that attribute in square brackets.
document.querySelector('p:first-child')     To select first-child of the paragraph tag.

// querySelectorAll --> Give a Node list. 
Have foreach prototype.
So we can use forEach function in this and can apply the thing which we want to apply on all the Node list items in that list. 
here we can Do as 
let myul = document.querySelectorAll('ul')
let myli = document.querySelectorAll('li')
myli.forEach( (item) => item.style.backgroundColor = "hotpink" )

As we know it was not an array it was a node list so we can't use map on it so if we want to use then we have to convert it 1st into array then can use the map on it.
